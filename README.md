# Project-Informational-Site

A Node.js learning project — a basic HTTP server built with only built-in modules (`http`, `fs/promises`) that serves static HTML pages.

```

The server runs on `http://localhost:3000` and serves the following routes:

| Route | Page |
|-------|------|
| `/` | Home |
| `/about` | About |
| `/contactme` | Contact |
| anything else | 404 |
```
